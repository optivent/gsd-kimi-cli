## [2.0.0] - 2025-02-04

### Added
- Initial stable release
- 27 GSD skills for project management
- 11 specialized GSD agents
- 9 reference knowledge bases
- Multi-agent workflow system
- Wave-based execution
- Verification built-in
- Status bar integration via patches
- Complete documentation

### Skills
- **Project**: gsd-new-project, gsd-progress, gsd-help
- **Phase**: gsd-plan-phase, gsd-execute-phase, gsd-verify-work
- **Milestone**: gsd-new-milestone, gsd-complete-milestone, gsd-audit-milestone
- **Management**: gsd-add-phase, gsd-insert-phase, gsd-remove-phase
- **Utilities**: gsd-debug, gsd-quick, gsd-check-todos, gsd-map-codebase

### Agents
- gsd-executor, gsd-planner, gsd-verifier, gsd-debugger
- gsd-roadmapper, gsd-phase-researcher, gsd-project-researcher
- gsd-research-synthesizer, gsd-codebase-mapper
- gsd-plan-checker, gsd-integration-checker
